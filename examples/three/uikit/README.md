# uikit game UI (Three.js)

A fullscreen game front-end composed with `@three-flatland/uikit` and its
shadcn-flavoured kit `@three-flatland/uikit-default`, sitting over a lit dungeon
room — a `TileMap2D` lit by two flickering `Light2D` torches. Tabs (Play /
Loadout / Settings), a live text `Input` for the player name, volume sliders, a
fullscreen switch, a vsync checkbox, a difficulty radio group, a scrollable
save-slot list, primary and secondary buttons, an animated loading `Progress`
bar, and lucide icons — all batched through Flatland's TSL-native pipeline. A
live FPS readout, ticking credits, and an animated stamina bar drive from the
render loop.

Part of [three-flatland](https://github.com/thejustinwalsh/three-flatland) — a TSL-native 2D rendering library for Three.js.

> Also available as a [React example](../../react/uikit).
