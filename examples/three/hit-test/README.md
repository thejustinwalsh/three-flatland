# Hit Test (Three.js)

Pointer picking with a plain `Raycaster` — no event library.

Hover highlights coins, clicking collects them, clicking empty ground walks the knight.
The knight has `hitTestMode = 'none'` so it is invisible to raycast traversal.

Part of [three-flatland](https://github.com/thejustinwalsh/three-flatland) — a TSL-native 2D rendering library for Three.js.

> Also available as a [React example](../../react/hit-test).
