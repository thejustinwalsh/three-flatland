**three-flatland** is a high-performance 2D sprite and effects library for [Three.js](https://threejs.org/) and [React Three Fiber](https://r3f.docs.pmnd.rs/), using WebGPU and TSL (Three Shader Language).

<Aside type="note" title="Early Alpha">
  three-flatland is in active development. We're exploring performant, maintainable, and extensible patterns for GPU-driven 2D rendering with WebGPU. The API will evolve as we refine these systems. [Follow along on GitHub](https://github.com/thejustinwalsh/three-flatland) â€” your feedback shapes what we build.
</Aside>

## Packages

| Package | Description |
|---------|-------------|
| `@three-flatland/core` | Core sprite system, materials, and rendering |
| `@three-flatland/nodes` | TSL shader nodes for custom effects |
| `@three-flatland/react` | React Three Fiber components and hooks |
| `@three-flatland/presets` | Pre-configured effect presets |

## Requirements

- **Three.js** >= 0.182.0 (required for TSL/WebGPU support)
- **React** >= 19.0.0 (for React packages)
- **@react-three/fiber** >= 10.0.0-alpha.0 (for React packages)
