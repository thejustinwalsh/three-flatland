**three-flatland** is a high-performance 2D sprite and effects library for [Three.js](https://threejs.org/) using WebGPU and TSL (Three Shader Language).

## Features

<CardGrid>
  <Card title="WebGPU Native" icon="rocket">
    Built from the ground up for WebGPU with TSL shaders for maximum performance.
  </Card>
  <Card title="Batch Rendering" icon="list-format">
    Automatic batching and instancing for rendering thousands of sprites efficiently.
  </Card>
  <Card title="React Three Fiber" icon="puzzle">
    First-class React Three Fiber integration with declarative JSX components.
  </Card>
  <Card title="Animation System" icon="forward-slash">
    Built-in sprite animation with spritesheet support and frame-based timing.
  </Card>
</CardGrid>

## Packages

| Package | Description |
|---------|-------------|
| `@three-flatland/core` | Core sprite system, materials, and rendering |
| `@three-flatland/nodes` | TSL shader nodes for custom effects |
| `@three-flatland/react` | React Three Fiber components and hooks |
| `@three-flatland/presets` | Pre-configured effect presets |

## Requirements

- **Three.js** >= 0.182.0 (required for TSL/WebGPU support)
- **React** >= 19.0.0 (for React packages)
- **@react-three/fiber** >= 10.0.0-alpha.0 (for React packages)

## Browser Support

three-flatland requires WebGPU support. Check [caniuse.com/webgpu](https://caniuse.com/webgpu) for current browser compatibility.
