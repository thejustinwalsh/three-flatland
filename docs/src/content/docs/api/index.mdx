## Packages

### @three-flatland/core

Core sprite system and rendering utilities.

- [Sprite2D](/api/sprite2d) - Basic 2D sprite
- [AnimatedSprite2D](/api/animated-sprite2d) - Animated sprite with spritesheet support
- [InstancedSprite2D](/api/instanced-sprite2d) - Instanced rendering for many sprites
- [Renderer2D](/api/renderer2d) - 2D rendering manager
- [Tilemap](/api/tilemap) - Tile-based map rendering

### @three-flatland/nodes

TSL shader nodes for custom effects.

- [outlineNode](/api/outline-node) - Outline effect
- [glowNode](/api/glow-node) - Glow effect
- [pixelateNode](/api/pixelate-node) - Pixelation effect
- [colorReplaceNode](/api/color-replace-node) - Color replacement

### @three-flatland/react

React Three Fiber components and hooks.

- [Components](/api/react-components) - JSX components
- [Hooks](/api/react-hooks) - Custom React hooks

### @three-flatland/presets

Pre-configured effect presets.

- [Pixel Art](/api/preset-pixel-art) - Retro pixel art effects
- [Neon](/api/preset-neon) - Neon glow effects
- [Particle](/api/preset-particle) - Particle system presets
