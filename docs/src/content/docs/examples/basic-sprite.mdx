## SpriteGroup

For batched rendering with many sprites, use `SpriteGroup`. Sprites added to a group are automatically batched by material:

```typescript
const spriteGroup = new SpriteGroup();
scene.add(spriteGroup);
spriteGroup.add(sprite);
```

For simple cases with few sprites, you can add them directly to the scene.

## Sprite2D

The `Sprite2D` class represents a 2D sprite. It requires a texture and supports optional properties like anchor point and size:

```typescript
const sprite = new Sprite2D({
  texture: myTexture,
  anchor: [0.5, 0.5],  // Center anchor
});
```
