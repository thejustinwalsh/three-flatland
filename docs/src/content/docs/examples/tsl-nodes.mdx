## Applying Effects

Use the `@three-flatland/nodes` package for pre-built effects:

```typescript
import { outlineNode, glowNode } from '@three-flatland/nodes';

sprite.material.colorNode = outlineNode({
  color: 0xff0000,
  width: 2,
});
```

## Available Nodes

| Node | Description |
|------|-------------|
| `outlineNode` | Solid colored outline |
| `glowNode` | Soft glow effect |
| `pixelateNode` | Retro pixelation |
| `colorReplaceNode` | Swap colors |

## Custom Nodes

Create custom effects using Three.js TSL functions:

```typescript
import { tslFn, vec4, uv, texture } from 'three/tsl';

const customEffect = tslFn(({ tex }) => {
  const color = texture(tex, uv());
  // Invert colors
  return vec4(1.0 - color.r, 1.0 - color.g, 1.0 - color.b, color.a);
});
```
