## Variant Explorations

Four parallel design directions for the BrandAsset compositions. Each variant
keeps the retro pixel-art icon and Silkscreen wordmark intact (those are the
established brand) but reimagines the surrounding canvas with a different
aesthetic vocabulary. Each variant is shown at the OG card size (1200×630),
the Bluesky banner size (3000×1000), and the icon-only size (512×512) so you
can judge how the language scales across aspect ratios.

### Variant A — Holographic Foil

*Layered conic gradients + chromatic-aberration ghosts on the wordmark + per-gem SVG specular filters in `<defs>`. Reads as a foil-stamped trading card; the wordmark literally is the foil.*
