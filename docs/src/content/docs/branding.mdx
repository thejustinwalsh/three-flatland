## Variant Explorations — gem-led, design-system-grounded

Seven parallel design directions, each anchored to a specific gem (or
gem-combo) from the design system's palette. Each variant iterated through
three revisions — **v1** (direct hit on the brief), **v2** (impeccable
critique + frontend-design fixes), **v3** (impeccable polish + per-aspect
adapt — the "shippable" version). All revisions render at the OG card
size (1200×630) so you can compare evolution within a variant + across
variants on common ground.

Each variant uses **only real `var(--<gem>)` tokens** from `theme.css`,
keeps the retro pixel-art icon and Silkscreen wordmark intact, and extends
existing site primitives (FeatureCard foil rim, sidebar section pip, TOC
active marker, StatsBanner gem underlines, Pagination per-destination
gem, aside type indicators).
