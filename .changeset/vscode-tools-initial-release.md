Initial publish-ready release of the Flatland Tools VS Code extension: FL Sprite Atlas (packing + animation editor, TexturePacker/Aseprite read+write), FL Image Encoder (PNG/WebP/AVIF/KTX2 comparison + encoding), FL Normal Baker (region editor for baking normal maps), FL Atlas Merge, and FL Audio (inline ZzFX/ZzFXM/Tone.js/Wad playback via CodeLens, plus the ZzFX Studio tuner panel). Packaged for both the VS Code Marketplace and Open VSX.
