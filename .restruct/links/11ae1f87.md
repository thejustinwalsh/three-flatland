## Context Rules
1. Always `orthographic` camera for 2D
2. `<Scene>` component inside Canvas (hooks like `usePane` require Canvas context)
3. Mutate refs directly — never `setState` in the render loop
4. Use `delta` from useFrame for time-based updates: `sprite.update(delta * 1000)`
5. Use `{ priority: N }` for ordering when needed

## Anti-Patterns
1. Import from `@react-three/fiber` — always use `@react-three/fiber/webgpu`
2. Use `setState` inside `useFrame` — mutate refs directly
3. Use GLSL or `onBeforeCompile` — this project uses TSL node materials
4. Use Web Awesome components — examples use `@three-flatland/tweakpane`
5. Use `Date.now()` for animation timing — use `state.clock.elapsedTime` or `delta`
6. Skip stats monitoring — every example must call `useStatsMonitor(stats)` (or equivalent) after `usePane()`
7. Destructure `useThree()` in hot paths — use individual selectors
8. Forget `extend()` — R3F won't recognize library classes without it
