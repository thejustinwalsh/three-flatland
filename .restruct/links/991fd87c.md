## Context Rules
1. `pnpm build` — turbo-orchestrated across all packages
2. `pnpm dev` — docs (port 4000) + examples MPA server (port 5174) via microfrontends proxy at http://localhost:5173
3. `pnpm --filter=example-react-tilemap dev` — run a single example
4. `pnpm typecheck` / `pnpm lint` / `pnpm test`
5. `pnpm syncpack:examples` / `pnpm syncpack:minis` — sync versions after changing catalog in pnpm-workspace.yaml
6. No semicolons, single quotes, trailing commas (Prettier)
7. `type` keyword required for type-only imports (`consistent-type-imports` + `verbatimModuleSyntax`)
8. Unused vars must be prefixed with `_`
9. WebGPU + TSL (Three Shader Language) exclusively — no WebGL, no GLSL
10. R3F examples import from `@react-three/fiber/webgpu`, not `@react-three/fiber`
11. Three.js users: `import from 'three-flatland'` — R3F users: `import from 'three-flatland/react'`
12. Shared versions in `pnpm-workspace.yaml` catalog; `pnpm.overrides` maps `@three-flatland/*` to `workspace:*`
13. Examples use Tweakpane (`@three-flatland/tweakpane/react`) for UI controls
14. Examples always exist in **pairs** — Three.js + React. Create both or neither.
15. `examples/three/` = plain Three.js, `examples/react/` = React Three Fiber
16. R3F classes must be registered with `extend()` before use in JSX
17. All Three.js objects used as R3F JSX elements need: optional constructor params, property setters, array-compatible setters

## Constraints
1. Performance is critical — minimize draw calls, batch sprites via SpriteGroup, watch frame budgets
2. All custom Three.js classes must work with R3F's no-arg construction + property-setting pattern

## Anti-Patterns
1. Use GLSL or `onBeforeCompile` — all shaders use TSL node materials
2. Use `WebGLRenderTarget` — use renderer-agnostic `RenderTarget`
3. Import from `@react-three/fiber` — always `@react-three/fiber/webgpu`
4. Use Web Awesome (`@awesome.me/webawesome`) — examples use Tweakpane now
5. Add `declare global { namespace JSX }` — use `ThreeElements` interface augmentation via `three-flatland/react`
