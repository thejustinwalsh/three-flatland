# @three-flatland/uikit-horizon

Horizon component kit for [`@three-flatland/uikit`](../uikit), based on Meta's
Reality Labs Design System (RLDS) — buttons, dropdowns, sliders, panels, and
input fields tuned for XR/spatial UI.

```ts
import { Button, Panel, Dropdown } from '@three-flatland/uikit-horizon'
```

```tsx
import { Button, Panel, Dropdown } from '@three-flatland/uikit-horizon/react'
```

## Forked from pmndrs/uikit
